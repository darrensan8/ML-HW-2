# CMPE188 HW2 — Neural Network Tasks
April 13, 2026

Four neural network tasks built with PyTorch following protocols in https://github.com/lkk688/CoderGym/tree/main.

## Tasks

| File            | Dataset               | New Feature |
|-----------------|-----------------------|-------------|
| `bagging-nn.py` | sklearn Wine          | Bagging ensemble of 10 MLPs with majority vote |
| `bCancer-nn.py` | sklearn Breast Cancer | MLP with Batch Normalization vs baseline |
| `diabetes-nn.py`| sklearn Diabetes      | MLP with Adam + StepLR learning rate scheduler |
| `digits-nn.py`  | sklearn Digits        | MLP with Dropout regularization vs baseline |
